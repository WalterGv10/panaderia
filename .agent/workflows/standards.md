# Project Creation & Development Standards

Adopt these principles for all web development tasks in this project.

## Core Technology Stack
- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Rendering**: SSR, SSG, or ISR as required for performance.

## Architectural Principles
- **Component-Based**: Use modular, reusable components.
- **Separation of Concerns**: Keep logic, styles, and structure organized.
- **Mobile-First**: Design for the smallest screens first, then scale up.

## SEO & Accessibility
- **Semantic HTML**: Always use `<section>`, `<header>`, `<footer>`, `<h1>-<h6>`, etc.
- **Metadata**: Comprehensive Meta tags, OpenGraph, and Twitter cards.
- **Structured Data**: JSON-LD for rich search results.

## Security & Best Practices
- **HTTPS**: Deployment must be over HTTPS.
- **Dependencies**: Keep dependencies safe and minimal.

## Content & UX
- **Clear Value Proposition**: Immediate clarity for the user.
- **Readable Structure**: Use proper hierarchy and spacing.
- **User Intent Focused**: Design around what the user wants to achieve.
